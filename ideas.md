# Brainstorm de Design - Portfólio Wagner Karoleski

## Contexto
Portfólio para Wagner Karoleski, QA Engineer e Analista de Sistemas com foco em automação e infraestrutura. Stack: Cypress, Docker, SQL Server, MySQL, BDD, TDD. Formação em DevOps. Experiência em SKA Automação e Atlas Technologies.

---

## <response>

### Ideia 1: **Technical Minimalism** (Probabilidade: 0.08)

**Design Movement:** Bauhaus meets Modern Tech — Funcionalismo rigoroso com elegância suíça.

**Core Principles:**
- Clareza extrema: cada elemento tem propósito claro
- Tipografia forte como protagonista visual
- Espaço negativo generoso para respiração
- Paleta monocromática com um accent color técnico

**Color Philosophy:**
- Fundo: Branco puro (`#FFFFFF`) ou off-white (`#F8F9FA`)
- Texto principal: Cinza escuro profundo (`#1A1A1A`)
- Accent: Azul elétrico (`#0066FF`) — representa tecnologia, confiabilidade
- Elementos secundários: Cinza neutro (`#E5E5E5`)
- Filosofia: Minimalismo transmite profissionalismo e controle técnico

**Layout Paradigm:**
- Grid assimétrico 3-coluna com hierarquia clara
- Seções empilhadas verticalmente com transições suaves
- Sidebar esquerda com navegação fixa (sticky)
- Conteúdo principal com largura máxima controlada
- Sem rounded corners excessivos — ângulos retos para rigor

**Signature Elements:**
1. Linhas horizontais finas como divisores (1px, cor accent)
2. Cards com border-left em accent color (4px)
3. Tipografia em escala geométrica (1.2 ratio)

**Interaction Philosophy:**
- Transições sutis (200ms ease-out)
- Hover: mudança de cor accent + underline sutil
- Focus states visíveis e acessíveis
- Sem animações desnecessárias — movimento intencional

**Animation:**
- Fade-in ao scroll (opacity 0→1, 300ms)
- Slide-in lateral para cards (transform translateX, 400ms)
- Hover: scale 1.02 + shadow sutil
- Sem parallax ou efeitos distrativos

**Typography System:**
- Display: IBM Plex Serif (bold, 3.2rem) — autoridade
- Heading: IBM Plex Sans (semibold, 1.8rem) — estrutura
- Body: IBM Plex Sans (regular, 1rem) — legibilidade
- Mono: IBM Plex Mono (regular, 0.875rem) — código/tech

## <response>

### Ideia 2: **Dark Tech Sophistication** (Probabilidade: 0.07)

**Design Movement:** Cyberpunk meets Enterprise Design — Escuro, profundo, com toques de neon.

**Core Principles:**
- Fundo escuro como canvas para conteúdo
- Contraste alto para legibilidade e impacto
- Efeitos de profundidade (blur, shadows)
- Paleta de cores saturadas e vibrantes

**Color Philosophy:**
- Fundo: Preto profundo com leve gradiente (`#0A0E27` → `#1A1F3A`)
- Texto principal: Branco puro (`#FFFFFF`)
- Accent primário: Ciano/Turquesa (`#00D9FF`) — tech, energia
- Accent secundário: Roxo (`#9D4EDD`) — inovação
- Filosofia: Escuro transmite profundidade técnica e modernidade

**Layout Paradigm:**
- Hero section full-width com gradiente diagonal
- Seções alternadas com background cards em camadas
- Grid 2-3 colunas com espaçamento assimétrico
- Elementos flutuantes com glassmorphism (backdrop-blur)
- Divisores com gradientes (não linhas sólidas)

**Signature Elements:**
1. Glassmorphic cards com border ciano (2px)
2. Gradientes diagonais em backgrounds
3. Ícones com glow effect (box-shadow neon)

**Interaction Philosophy:**
- Transições fluidas e responsivas (300ms)
- Hover: glow effect + cor accent intensifica
- Click feedback com pulse animation
- Micro-interactions que reforçam tech aesthetic

**Animation:**
- Entrance: fade-in + slide-up (400ms cubic-bezier)
- Hover: glow animation (box-shadow pulsante)
- Scroll: parallax sutil em hero (20px offset)
- Loading states: spinner com gradiente animado

**Typography System:**
- Display: Space Mono (bold, 3.5rem) — futurista
- Heading: Space Mono (semibold, 2rem) — tech
- Body: Inter (regular, 1rem) — legibilidade moderna
- Mono: Fira Code (regular, 0.875rem) — código com estilo

## <response>

### Ideia 3: **Warm Professional Craft** (Probabilidade: 0.09)

**Design Movement:** Artisanal meets Modern — Quente, acessível, com textura e humanidade.

**Core Principles:**
- Paleta quente que transmite confiança e acessibilidade
- Tipografia generosa e confortável
- Texturas sutis (ruído, padrões) para dimensão
- Espaçamento generoso e respiração visual

**Color Philosophy:**
- Fundo: Creme quente (`#FAF8F3`)
- Texto principal: Marrom escuro (`#2D2520`)
- Accent primário: Laranja-terra (`#D97706`) — confiança, energia
- Accent secundário: Verde-sálvia (`#6B8E71`) — crescimento
- Filosofia: Cores quentes transmitem humanidade e profissionalismo acessível

**Layout Paradigm:**
- Seções com fundo alternado (creme + cards bege)
- Asymmetric grid com imagens grandes
- Sidebar com navegação contextual
- Elementos com rounded corners generosos (12-16px)
- Divisores como padrões sutis (não linhas)

**Signature Elements:**
1. Cards com shadow suave e border-radius (14px)
2. Ícones com background circular em accent color
3. Tipografia em escala generosa com line-height alto

**Interaction Philosophy:**
- Transições suaves e naturais (250ms ease-in-out)
- Hover: mudança de cor + elevação (shadow)
- Focus states com outline em accent
- Feedback tátil sem ser agressivo

**Animation:**
- Entrance: fade-in suave (300ms)
- Scroll: fade-in + slide-up (350ms ease-out)
- Hover: scale 1.05 + shadow elevation
- Sem efeitos distrativos — movimento proposital

**Typography System:**
- Display: Crimson Text (bold, 3.2rem) — elegância
- Heading: Crimson Text (semibold, 2rem) — autoridade
- Body: Lato (regular, 1.05rem) — legibilidade quente
- Mono: JetBrains Mono (regular, 0.875rem) — código acessível

---

## Decisão Final

**Escolhido: Dark Tech Sophistication**

Justificativa: Para um QA Engineer e DevOps specialist, a estética dark tech comunica profundidade técnica, modernidade e inovação. O contraste alto garante legibilidade, os efeitos de profundidade (glassmorphism, glow) transmitem sofisticação, e a paleta ciano + roxo é reconhecida no universo tech como símbolo de expertise. O layout assimétrico e os gradientes diagonais criam movimento visual sem ser distrativo. Esta abordagem posiciona Wagner como um profissional moderno, atualizado com tendências 2026.
